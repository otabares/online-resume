Before you start exploring, let me tell you why I started Udacity's Intro to Programming Nanodegree.

I took my first web design class in 2002 when I was in tenth grade. At that time, XHTML was still the new kid in town, and CSS was just starting to gain popularity. Shortly after, I took two programming classes: QBasic and AP Computer Programming (Java). I struggled with Java and ended up dropping it. This experience left a bitter taste and led me to avoid programming for several years.

In 2008, I took another Java programming class and did well. It was fun, but nothing else. Four years later, in 2012, I took an edX course to dabble further into the world of computer science. This course was 6.00x, MIT's Intro to Computer Science and Programming Using Python. It was intense. It was fun. It was frustrating. I was not ready yet.

It's now been 15 years since I started exploring the wonderful world of computer science, and I'm finally ready. I'm ready to commit: that is why I started this nanodegree.

Why did I start with Intro to Programming? I started with IPND because I want to ensure I close any potential gaps. However, I do not plan to stop with IPND. I'm looking forward to Full Stack Web Development, Become an iOS Developer, and beyond. In fact, my plan is to use this nanodegrees as preparation for Georgia Tech's OMSCS.

I'm ready, and I'm excited for what the future holds.
